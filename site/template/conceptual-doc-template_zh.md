# 概念文档模板

请提供您介绍的系统/概念/特性的一般信息。

## 提供尽可能多的相关信息

以便用户理解。在必要时使用[超链接](页面的网址)。

## 在参考文档中使用简洁明了的句子

以使用户更易理解。

## 使用表格比较事物或提供分类信息

|              | 第一项     | 第二项      |
| ------------ | ---------- | ----------- |
| 差异 A       |            |             |
| 差异 B       |            |             |



## 使用列表提供顺序信息

这是一个无序列表：

- 陈述 1
  - 子陈述 1
  - 子陈述 2
- 陈述 2
- ...

这是一个有序列表：

1. 第一件事
2. 第二件事
3. ...
