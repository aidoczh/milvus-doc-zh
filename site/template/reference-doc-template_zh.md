# 参考文档模板

请提供您详细阐述的参考文档的一般信息。

## 参考文档应当全面准确

因为它是其他文档的基础和真实来源。

## 在参考文档中使用简洁明了的句子

以便让用户更容易理解。

## 使用表格比较事物或提供分类信息

|              | 第一项     | 第二项      |
| ------------ | ---------- | ----------- |
| 差异 A       |            |             |
| 差异 B       |            |             |



## 使用列表提供顺序信息

这是一个无序列表：

- 陈述 1
  - 子陈述 1
  - 子陈述 2
- 陈述 2
- ...

这是一个有序列表：

1. 第一件事
2. 第二件事
3. ...
