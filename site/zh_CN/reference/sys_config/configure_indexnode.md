# 索引节点相关配置

本主题介绍了 Milvus 的索引节点相关配置。

索引节点为向量构建索引。

在本节中，您可以配置索引节点端口等。

  <thead>
    <tr>

    </tr>

      <td>21121</td>
    </tr>

</table>
